# Zero8 Slim-Surface MX Keycap

Choc v2(Lofree Flow)に対応した15.5mm幅のキーキャップです。  
主に0.8u(16mmピッチ)以上のキーボードに取り付けることが可能です。  
天面が平坦なため、3Dプリンタでの印刷が容易に行えます。
